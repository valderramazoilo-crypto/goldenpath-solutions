GoldenPath — Degular fonts (self-hosted)
=========================================

Drop your licensed Degular .woff2 files in THIS folder with these exact names.
The site already references them and will switch from the Hanken Grotesk
fallback to Degular automatically — no code changes needed.

Required (body weights):
  Degular-Light.woff2       (300)
  Degular-Regular.woff2     (400)
  Degular-Medium.woff2      (500)
  Degular-Semibold.woff2    (600)
  Degular-Bold.woff2        (700)

Optional (oversized display headings):
  Degular-Display-Semibold.woff2  (600)

Tip: if you only have .otf/.ttf, convert to .woff2 (e.g. with
https://transfonter.org or `fonttools`) for the best web performance.
Until these files exist, headings/body render in Hanken Grotesk, which is
visually close so the layout stays correct.
